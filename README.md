# ✨ Ece'ye Özel Masallar

Sevgilim Ece için yapılmış, büyülü bir masal sitesi.

## Özellikler

- Yıldızlı gece gökyüzü ve büyülü orman teması
- Kayan yıldız animasyonları ve ateşböcekleri
- Sayfa çevirme efektli kitap okuyucu
- Klavye (← →) ve mobil kaydırma desteği

## Yeni Masal Eklemek

`index.js` dosyasının başındaki `STORIES` dizisine ekle:

```js
{
    id: 2,
    title: "Masal Adı",
    icon: "🌟",
    coverColor: "#1a0a3e",
    content: `Masal metni...`
}
```

## Kullanılan Teknolojiler

HTML · CSS · Vanilla JavaScript · Google Fonts (Cinzel, Crimson Text, Dancing Script)
